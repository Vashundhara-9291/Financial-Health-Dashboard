# CodeAlpha_Financial-Health-Dashboard
A Financial Health Dashboard built in Power BI provides a clear view of SME performance by visualizing key metrics such as revenue, expenses, profit margins, and cash flow. It uses interactive charts and filters to highlight top-performing companies, track financial trends, and identify risks. With a modern design and color-coded visuals.
